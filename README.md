# AutoHotkey 自动按键脚本

一个功能强大的 AutoHotkey v2.0 脚本，支持多按键自动点击，带有智能防封机制和实时调速功能。

## 功能特性

- 🎮 **多按键支持**：同时管理多个按键的自动点击
- ⚡ **智能防封**：二阶随机算法，模拟人类按键行为
- 🎛️ **实时调速**：按住按键时使用 +/- 键调整速度
- 🖥️ **OSD 显示**：实时显示运行状态和按键配置
- 💾 **配置持久化**：自动保存按键配置到文件

## 使用方法

### 基本操作

- **F12**：全局暂停/恢复
- **Ctrl+Alt+Shift+Key**：开启/关闭指定按键的自动点击
- **按住按键时按 +/-**：调整该按键的点击间隔（±100ms）

### 支持的按键

- 字母键：A-Z
- 数字键：0-9
- 功能键：F1-F12
- 符号键：`[`, `]`, `;`, `'`, `,`, `.`, `/`, `-`, `=`

## 配置说明

脚本会在同目录下创建 `config.ini` 文件保存按键配置，该文件已加入 `.gitignore`，不会上传到仓库。

## 系统要求

- Windows 操作系统
- AutoHotkey v2.0 或更高版本
